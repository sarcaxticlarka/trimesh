# 🧊 TriMesh Client

The frontend for the TriMesh 3D Model Marketplace, built with Next.js 15.

## 🚀 Features
- **Modern UI**: Glassmorphic design with GSAP animations.
- **AI Studio**: Powered by Groq & Cloudflare for asset generation.
- **Dynamic Browsing**: Categorized model gallery with real-time filters.
- **User Workspace**: Personalized dashboard for uploads and bookmarks.

## 🛠 Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS
- GSAP
- Lucide Icons
- Axios

## ⚙️ Setup
1. `npm install`
2. Create `.env.local`:
   ```env
   NEXT_PUBLIC_API_URL="http://localhost:5000"
   ```
3. `npm run dev`
